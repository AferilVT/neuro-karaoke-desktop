# Assets

Place application icons here:

- `icon.png` - 512x512 PNG for Linux
- `icon.ico` - Windows icon (256x256 recommended)
- `icon.icns` - macOS icon

You can generate these from a single PNG using online tools or:
- https://www.icoconverter.com/ (for .ico)
- https://iconverticons.com/online/ (for .icns)

For now, the application will use a default Electron icon if these are not provided.
